React Grid - Band Columns

The Grid component supports band columns. You can add this feature by specifying banded columns for a band column.

Related Plugins

The following plugins implement the Band Columns feature:

  • TableHeaderRow - renders the banded cells
  • TableBandHeader - renders the band cells

Basic Setup

Import the plugins listed above and specify the TableBandHeader pluginâ€™s columnBands property to set up a Grid with band columns. The nesting level depth is unlimited.
